# 🚀 Machine Learning Model Deployment using Flask

## 📌 Overview
This project demonstrates deploying a machine learning model using Flask. The model is saved as `fish_market_model.pkl` and served via a REST API.

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- Joblib
- NumPy & Pandas
